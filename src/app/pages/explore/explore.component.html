VASR dataset contains several analogies types. In this section, you can explore the variety of them. By choosing an option from the suggestions below, a suitable analogy will be displayed.

For example, if choosing 'Agent', an analogy with the main difference between A and A' being 'Agent' will be shown. One should select the right B' so that the main difference between B and B' will be 'Agent'.
